# BrowserTabRecorder

BrowserTabRecorder - расширение Chrome (Manifest V3) для записи активной вкладки браузера в файл WebM.

## Что умеет

- Записывает видео и звук активной вкладки.
- Может добавлять микрофон в запись (если выдали доступ).
- Показывает таймер и размер записи в popup.
- Позволяет вручную ставить запись на паузу и возобновлять.
- Сохраняет результат через системный диалог `Save As`.
- При достижении лимитов (5 часов или 10 ГБ) предлагает:
  - продолжить запись,
  - или остановить и сохранить.
- Если вкладка закрыта во время записи, предлагает сохранить частичный файл.

## Установка

### Требования

- Google Chrome 116+
- Node.js 16+ (только для генерации иконок, если нужно)

### Шаги

1. (Опционально) Сгенерируйте иконки:

```bash
node scripts/generate-icons.js
```

2. Откройте `chrome://extensions/`.
3. Включите `Developer mode`.
4. Нажмите `Load unpacked`.
5. Выберите папку проекта `BrowserTabRecorder`.

## Как работает

- `popup/` - интерфейс управления записью.
- `service-worker.js` - координирует состояние, маршрутизирует сообщения между компонентами.
- `offscreen/offscreen.js` - захват медиапотока, микширование аудио, запись через `MediaRecorder`.
- `content/content-script.js` - проверка разрешений микрофона.
- `lib/audio-mixer.js` - микширование аудио вкладки и микрофона.

Пайплайн сохранения:

1. В начале записи popup открывает `Save As` и получает `FileSystemFileHandle`.
2. После остановки offscreen собирает итоговый `Blob`.
3. `Blob` передается через IndexedDB (а не напрямую через IPC).
4. Popup читает `Blob` из IndexedDB и записывает в выбранный файл.

## Использование

1. Перейдите на вкладку, которую нужно записать, и сделайте ее активной.
2. Нажмите иконку расширения.
3. Нажмите `Start Recording` и выберите путь сохранения.
4. При необходимости подтвердите добавление микрофона.
5. При необходимости используйте `Pause` / `Resume`.
6. Для завершения нажмите `Stop & Save`.

## Ограничения

- Записывается только активная вкладка.
- На время записи popup должен оставаться открытым.
- Формат вывода: WebM.
